## 说明

自定义英雄打对战和战役系列视频。技能设计主要来源为评论区。

## 环境部署

系统/软件要求：
- Windows 11 x64
- 重制版2.0.4
- [SharpForge](https://github.com/yatyricky/SharpForge)
- Git
